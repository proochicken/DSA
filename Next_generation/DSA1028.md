# Describe:

Cho dãy số A[] có N phần tử. Hãy liệt kê tất cả các tổ hợp chập K của tập các phần tử khác nhau trong A[]. Các tổ hợp cần liệt kê theo thứ tự từ điển (tức là trong mỗi tổ hợp thì giá trị từ nhỏ đến lớn, và tổ hợp sau lớn hơn tổ hợp trước).


Input

Dòng đầu ghi hai số N và K.

Dòng thứ 2 ghi N số của mảng A[]. Các giá trị không quá 1000.

Dữ liệu đảm bảo số phần tử khác nhau của A[] không quá 20 và K không quá 10.

Output

Ghi ra lần lượt các tổ hợp tìm được, mỗi tổ hợp trên một dòng.


Ví dụ

```text
Input

8 3
2 4 4 3 5 1 3 4
```

```text
Output

1 2 3
1 2 4
1 2 5
1 3 4
1 3 5
1 4 5
2 3 4
2 3 5
2 4 5
3 4 5
```


# Solution:

```C++
#include <bits/stdc++.h>
using namespace std;
int n, k, a[25];
bool check = false;
int idx[25];
set<int> se;
void nhap(){
    cin >> n >> k;
    for(int i = 1; i <= n; i++){
        int x; cin >> x;
        se.insert(x);
    }
    int cnt = 1;
    for(int x : se){
        a[cnt++] = x;
    }
    for(int i = 1; i <= k; i++){
        idx[i] = i;
    }
    n = se.size();
}
void sinh(){
    int i = k;
    while(i >= 1 && idx[i] == n - k + i) i--;
    if(i == 0) check = true;
    else{
        idx[i]++;
        for(int j = i + 1; j <= k; j++){
            idx[j] = idx[j- 1] + 1;
        }
    }
}
int main(){
    nhap();
    while(check == false){
        for(int i = 1; i <= k; i++){
            cout << a[idx[i]] << " ";
        }
        cout << endl;
        sinh();
    }
}
```
