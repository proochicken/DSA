# Describe: 

Hãy in ra tất cả các xâu nhị phân độ dài N, có K bit 1 theo thứ tự từ điển tăng dần.


Input: Dòng đầu tiên là số lượng bộ test T (T ≤ 20). Mỗi test gồm 2 số nguyên N, K (1 ≤ K ≤ N ≤ 16).


Output:  Với mỗi test, in ra đáp án tìm được, mỗi xâu in ra trên một dòng.


Ví dụ:
 

```text
Input

2
4 2
3 2
 ```

```text
Output

0011
0101
0110
1001
1010
1100
011
101
110
```

# Solution:

```C++
#include <bits/stdc++.h>
using namespace std;
int n, k;
int a[20];
bool check = false;
void nhap(){
    cin >> n >> k;
    for(int i = 1; i <= n; i++){
        a[i] = 0;
    }
}
void sinh(){
    int i = n;
    while(i >= 1 && a[i] == 1){
        a[i] = 0;
        i--;
    }
    if(i == 0) check = true;
    else{
        a[i] = 1;
    }
}
int main(){
    int t; cin >> t;
    while(t--){
        nhap();
        check = false;
        while(check == false){
            int cnt = 0;
            for(int i = 1; i <= n; i++){
                if(a[i] == 1) cnt++;
            }
            if(cnt == k){
                for(int i = 1; i <= n; i++){
                    cout << a[i];
                }
                cout << endl;
            }
            sinh();
        }
    }
}
```
